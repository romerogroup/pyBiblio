pyBiblio documentation
====================================
pyBiblio is a Python library providing a set of functions to compute basic bibliometric measures. It takes in a Pandas dataframe parsed from Web of Science text files.

Instructions to get WoS files and dataframes can be found in the :ref:`label-wos` section.
